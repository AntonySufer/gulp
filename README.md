# gulp
gulp压缩css js img  


 安装del包
   npm install --save-dev gulp del

 全局安装   gulp
  npm install gulp --save-g

若全局安装不能运行，项目再安装一遍 
  npm install gulp --save-dev

压缩执行  见 package.json 包

   npm run clean  //删除所有dist下文件 ，清理

   npm run gulp   //执行default任务（包含js/css/img）



如项目不能运行可能npm包丢失，运行

  npm  install
